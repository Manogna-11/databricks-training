# Week 1 - Day 1

## Objective

The main goal of this session was to get familiar with basic SQL queries and understand how data is stored and retrieved from tables.

## Tasks Performed

* Created tables using the given SQL schema
* Ran different types of queries on the tables
* Checked outputs to make sure the queries were working correctly

## Topics Covered

* Basic SELECT statements
* Filtering using WHERE clause
* Sorting results using ORDER BY
* Aggregate functions like COUNT, SUM, AVG
* String matching using LIKE
* Working with date conditions

## Platform Used

* DB Fiddle (online SQL editor)

## Files Included

* schema.sql – for creating tables
* queries_sql.sql – contains all the SQL queries practiced
* output.txt – includes some query results

## Learning Outcome

Learned how to write simple SQL queries and understood how different clauses are used together. Also got basic idea about joins and aggregate functions.

## Challenges Faced

At first, it was a bit confusing to use WHERE conditions and combine multiple clauses, but after trying a few queries, it became easier to understand.

## Conclusion

Completed the tasks for Day 1 and gained basic knowledge of SQL. This helped in building confidence to solve more complex queries in the future.
