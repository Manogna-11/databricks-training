# SQL Joins Assignment

## Objective

To understand and implement different types of SQL JOIN operations including LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, and CROSS JOIN.

## Tasks Performed

- Retrieved student and course enrollment details
- Identified courses without enrollments
- Displayed instructor and course mapping
- Used LEFT JOIN and RIGHT JOIN queries
- Implemented FULL OUTER JOIN using UNION
- Generated reports combining multiple tables
- Practiced CROSS JOIN queries

## Topics Covered

- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- UNION
- NULL filtering
- Multi-table joins

## Tools Used

- MySQL
- DB-Fiddle / SQL Compiler

## Outcome

Successfully implemented SQL join operations to retrieve matched and unmatched records from multiple related tables.
